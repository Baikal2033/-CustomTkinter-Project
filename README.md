## 📋 О проекте

**Панель управления** — это десктопное приложение на Python с графическим интерфейсом CustomTkinter, которое объединяет в себе самые необходимые инструменты для управления системой Windows. Забудьте о поиске нужных настроек в разных меню — всё в одном окне!

Проект создан для быстрого доступа к основным функциям системы: от смены обоев до мониторинга производительности. Современный дизайн, плавные анимации и две темы оформления делают использование приятным и удобным.

## 🎯 Функции

### 🖼️ 1. Смена обоев рабочего стола
Мгновенная смена обоев рабочего стола в один клик. Выберите изображение из предустановленных вариантов — и фон рабочего стола изменится моментально.

- Поддержка форматов: `.jpg`, `.png`, `.bmp`
- Сохранение настроек после перезагрузки
- Стиль: **Заполнение** (оптимально для любых разрешений)

### 🗑️ 2. Очистка корзины
Быстрая и безопасная очистка корзины без лишних окон подтверждения.

- Очистка в один клик
- Без всплывающих окон
- Без звукового сопровождения

### 📊 3. Мониторинг системы
Отслеживание загрузки процессора, оперативной памяти и дискового пространства в реальном времени.

- ⚙️ **Процессор**: загрузка в %, количество ядер и потоков
- 🧠 **Оперативная память**: всего / использовано / свободно
- 💾 **Диски**: все разделы, свободное место, тип файловой системы

### 🚀 4. Тест скорости интернета
Встроенный спидтест для проверки скорости интернет-соединения. Загружает тестовый файл и вычисляет реальную скорость.

- 📥 Скорость загрузки (Мбит/с)
- 📦 Размер скачанного файла
- ⏱️ Время загрузки
- 🖥️ Информация о сервере

### ⏰ 5. Таймер выключения ПК
Планирование автоматического выключения компьютера через заданный промежуток времени.

- Быстрые пресеты: 30 мин, 1 час, 2 часа
- Возможность отмены таймера
- Понятный обратный отсчёт

### 🧹 6. Очистка кеша
Глубокая очистка системы от временных файлов и кеша браузеров.

- 🗑️ Запуск стандартной очистки диска Windows
- 📁 Очистка папок `Temp` (пользовательская и системная)
- 🌐 Очистка кеша **Google Chrome**
- ⚠️ Безопасно: важные файлы не затрагиваются

### 🎨 7. Две темы оформления
Переключение между тёмной и светлой темой в реальном времени.

- 🌙 **Тёмная тема** — стильный зелёный акцент
- ☀️ **Светлая тема** — яркий оранжевый акцент
- Мгновенное переключение всех элементов
